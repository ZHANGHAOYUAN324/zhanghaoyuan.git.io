Weather Data Imputation Using Deep Learning

Type: Time Series Data Processing Date: 2025

Developed a neural network architecture to recover missing daily weather measurements from a corrupted dataset spanning multiple decades. This project combined extensive data visualization, PyTorch-based data loading pipelines, and a custom deep learning model to accurately fill in missing values, resulting in a fully restored dataset ready for further meteorological analysis.

  • Implemented data exploration and visualization techniques, creating detailed line plots and histograms to compare weather variables before and after corruption, revealing the distribution and nature of missing values
  • Constructed PyTorch data pipelines, building TensorDatasets for paired corrupted and uncorrupted training data, alongside a separate DataLoader for test data with missing values, ensuring efficient batch processing and model training
  • Designed and trained a custom imputation network that learns temporal patterns in the data, accurately inferring and restoring missing measurements
  • Employed careful hyperparameter tuning and loss function selection to maximize imputation accuracy; generated a final test_set_nogaps.csv conforming to the original data structure
Python PyTorch Time Series Neural Networks Data Visualization
